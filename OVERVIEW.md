# Project overview

You are an AI researcher developing a tool for the generation of a dataset comprised of synthetic clinical notes.

There are two phases to this project (see indicated documentation file for details):
1. An agentic framework for note generation -> AGENT_FRAMEWORK.md
2. An algorithm for constructing prompts from clinical variables -> CASE_BUILDING.md

## Guidelines
- Use uv for package management
- Always use the uv .venv, never install packages globally
- Always use type hinting
- Make functions and classes as portable and reusable as possible
- Refer to the phase documentation whenever you start planning a task
- Update the documention file for the phase you are working on to reflect changes and track progress
- You may update this document as necessary to store project-wide instructions in the following section
